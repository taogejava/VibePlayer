六大核心功能

打开即是绚丽首页，一键直达任何媒体内容

🎵

听音乐

选择本地音乐文件夹，黑胶唱片旋转，粒子特效与频谱同步律动。支持 MP3、FLAC、WAV 等 10 种格式。

🎬

看视频

扫描本地视频文件，完整播放器控件。支持全屏、倍速(0.5x~2x)、快进快退、11 种视频格式。

📺

B站

粘贴 B站视频链接，自动解析并在播放器内嵌播放。告别跳转浏览器，沉浸式体验。

🔗

链接播放

粘贴任意音视频 URL 直接播放。自动检测文件类型，音频加入列表，视频全屏播放。

☁️

WebDAV

连接群晖、威联通、NextCloud 等网盘服务，直接浏览并播放音视频文件。

📦

AList

一次对接，访问百度网盘、阿里云盘、123云盘、蓝奏云、夸克等所有已挂载存储。

三步开始使用

为普通用户设计，无需任何技术基础

1

下载安装包

点击上方按钮，根据系统选择 Windows 安装包或 macOS DMG 文件。

2

安装 / 打开

Windows：运行 .exe 按提示安装
macOS：打开 DMG，将应用拖入「应用程序」文件夹。

3

选择媒体开始播放

打开即见绚丽首页，点击功能卡片或顶部标签，选择本地文件或连接网盘即可 🎵
