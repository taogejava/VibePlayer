核心功能

每一个细节都为沉浸感而生

🌌

粒子背景特效

Canvas 实时渲染浮动粒子，播放时自动生成，带发光拖尾效果，颜色随歌曲主题变化。

💿

黑胶唱片动画

播放时自动旋转，暂停时静止，配有锥形渐变纹理和动态光晕，复古与现代完美结合。

📊

频谱可视化

32 条彩色频谱条实时律动，响应音频频率变化，视觉节奏感十足。

🎤

歌词同步高亮

当前行渐变发光高亮，歌词自动滚动跟随播放进度，沉浸式卡拉 OK 体验。

📁

本地音乐库

选择本地文件夹，自动递归扫描，支持 MP3、FLAC、WAV、AAC、M4A 等 10 种格式。

🎬

哔哩哔哩内嵌播放

粘贴 B 站链接，自动解析视频信息，内嵌播放器直接观看，告别跳转切换。

📋

播放列表管理

收藏喜爱的歌曲，一键切换顺序/随机/单曲循环，完整播放控制体验。

🖥️

桌面原生应用

基于 Electron 打包，支持 Windows & macOS，双击即用，无需安装 Node.js 或浏览器。

三步开始使用

为普通用户设计，无需任何技术基础

1

下载安装包

点击上方按钮，根据系统选择 Windows 安装包或 macOS DMG 文件。

2

安装 / 打开

Windows：运行 .exe 按提示安装
macOS：打开 DMG，将应用拖入「应用程序」文件夹。

3

选择音乐开始播放

点击「本地」→「选择文件夹」，选中你的音乐目录，即可开始享受 🎵
